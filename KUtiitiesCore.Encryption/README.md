# KUtilitiesCore.Encryption

Servicios de seguridad para el cifrado y hashing de datos, proporcionando una capa de abstracción sencilla para operaciones criptográficas comunes.

## Características

- **Cifrado AES**: Implementación de cifrado simétrico avanzado para protección de datos sensibles.
- **DPAPI**: Integración con la Data Protection API de Windows para cifrado basado en el usuario/máquina.
- **Hashing**: Servicios de hashing para generación de huellas digitales de datos y validación de integridad.
- **Base64**: Utilidades para la codificación y decodificación segura de datos cifrados.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.Encryption`
