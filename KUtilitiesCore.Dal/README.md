# KUtilitiesCore.Dal

Capa de acceso a datos (Data Access Layer) optimizada para el trabajo con SQL Server, proporcionando una estructura robusta para la persistencia de información.

## Características

- **Unit of Work & Repositorios**: Implementación de patrones para garantizar la consistencia de los datos y el desacoplamiento de la lógica de negocio.
- **Inserciones Masivas (Bulk Insert)**: Soporte optimizado para la carga de grandes volúmenes de datos.
- **Gestión de Transacciones**: Control preciso de transacciones para asegurar la integridad de las operaciones.
- **Helpers de Procedimientos Almacenados**: Herramientas para simplificar la ejecución y gestión de SPs en SQL Server.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.Dal`
