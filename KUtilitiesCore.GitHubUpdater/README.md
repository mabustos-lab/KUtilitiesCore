# KUtilitiesCore.GitHubUpdater

Herramientas para la gestión de actualizaciones automáticas de aplicaciones y reporte de incidencias integradas con la API de GitHub.

## Características

- **Gestor de Actualizaciones**: Implementación completa para buscar y descargar nuevas versiones de la aplicación desde GitHub Releases.
- **Descarga de Assets**: Soporte para la selección y descarga optimizada de archivos adjuntos en los releases.
- **Reporte de Incidencias**: Integración para enviar reportes de errores directamente a los issues del repositorio de GitHub.
- **Análisis de Versiones**: Parser de versiones compatible con estándares semánticos para determinar si existen actualizaciones disponibles.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.GitHubUpdater`
