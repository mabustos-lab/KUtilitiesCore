# KUtilitiesCore

Librería de utilidades base para .NET, diseñada para proporcionar herramientas comunes que agilizan el desarrollo de aplicaciones robustas y mantenibles.

## Características

- **Validación de Datos Avanzada**: Marco de validación con atributos personalizados y constructores de reglas.
- **Conversión de Tipos**: Sistema flexible para manejar conversiones de datos complejas y tipos anulables.
- **Extensiones Ampliadas**: Gran variedad de métodos de extensión para LINQ, colecciones, strings y más.
- **Telemetría y Tracking**: Soporte integrado para métricas y seguimiento de cambios en entidades.
- **MVVM Base**: Estructuras fundamentales para soportar el patrón Model-View-ViewModel.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore`
