# KUtilitiesCore.MVVM

Implementaciones core para el patrón Model-View-ViewModel (MVVM), proporcionando las herramientas necesarias para desacoplar la lógica de negocio de la interfaz de usuario.

## Características

- **RelayCommand**: Implementación flexible de comandos para vincular acciones de la vista con la lógica del ViewModel.
- **ViewModel Helpers**: Clases base y extensiones para simplificar la implementación de notificaciones de cambios de propiedades.
- **ActionResult**: Sistema para manejar los resultados de las operaciones de los ViewModels de manera estandarizada.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.MVVM`
