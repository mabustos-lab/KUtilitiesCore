# KUtilitiesCore.Data

Utilidades especializadas para la manipulación, importación y exportación de datos, facilitando el intercambio de información entre diferentes formatos de archivo.

## Características

- **Importación de Datos**: Soporte avanzado para la lectura y carga de datos desde archivos CSV y Excel.
- **Exportación de Datos**: Herramientas para generar reportes y exportar conjuntos de datos a formatos CSV y Excel.
- **Gestión de Fuentes**: Abstracciones para manejar diversas fuentes de datos externas y lectura de archivos de disco.
- **Validación de Importación**: Integración con el sistema de reglas de validación para asegurar la calidad de los datos importados.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.Data`
