# KUtilitiesCore.MVVM.Messaging

Sistema de mensajería desacoplado diseñado para facilitar la comunicación entre componentes en aplicaciones que utilizan el patrón MVVM.

## Características

- **Messenger**: Implementación de un bus de mensajes ligero para enviar y recibir notificaciones entre ViewModels sin dependencias directas.
- **Mensajes Tipados**: Soporte para la definición de mensajes personalizados para asegurar la integridad de los datos transmitidos.
- **Suscripciones Flexibles**: Capacidad de registrar manejadores de eventos para tipos específicos de mensajes.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.MVVM.Messaging`
