# KUtilitiesCore.Logger

Servicios de registro (logging) flexibles y extensibles para aplicaciones .NET, permitiendo la captura de eventos en múltiples destinos.

## Características

- **Proveedores Múltiples**: Soporte para registro en archivos, bases de datos SQL Server y ventana de depuración.
- **Logger Compuesto**: Capacidad de enviar logs a múltiples destinos simultáneamente.
- **Información de Llamada**: Captura automática de información del contexto de llamada (clase, método, línea).
- **Niveles de Log**: Gestión de severidad para filtrar la importancia de los eventos registrados.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.Logger`
