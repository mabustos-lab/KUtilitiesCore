# KUtilitiesCore.DataAccess

Definiciones genéricas y abstracciones para la implementación de capas de persistencia, permitiendo la creación de arquitecturas de datos flexibles y desacopladas.

## Características

- **Patrón Specification**: Implementación del patrón de especificaciones para definir filtros de consulta de forma reutilizable y desacoplada.
- **Soporte de Paginación**: Infraestructura para la gestión de resultados paginados en consultas de datos.
- **Interfaces Base**: Definiciones estándar para Repositorios y Unit of Work, facilitando la implementación de múltiples proveedores de datos.
- **Abstracciones de Persistencia**: Herramientas para separar la lógica de acceso a datos de la implementación concreta.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.DataAccess`
