# KUtilitiesCore.Data.Win

Componentes de interfaz de usuario para Windows Forms destinados a facilitar la configuración de procesos de importación y exportación de datos.

## Características

- **Import Wizard**: Interfaz guiada para el proceso de importación de datos, permitiendo la definición de mapeos y reglas de validación.
- **Controles de Configuración**: Componentes visuales para la configuración de parámetros de lectura de archivos CSV y Excel.
- **Integración Visual**: Diseñado para integrarse fácilmente en aplicaciones Windows Forms existentes.

## Instalación

Instale la librería a través de NuGet:
`dotnet add package KUtilitiesCore.Data.Win`
